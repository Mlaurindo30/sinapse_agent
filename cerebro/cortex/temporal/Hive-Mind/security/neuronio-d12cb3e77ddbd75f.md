# Auditoria 100% real de código

O Conselho exige que a auditoria de código seja 100% real, sem simulações. Modelos não devem fazer suposições sem testar na infraestrutura real.

## Evidência (Groundedness)
> O Conselho exige auditoria 100% real de código. Modelos não devem supor nada sem testar na infraestrutura real.
